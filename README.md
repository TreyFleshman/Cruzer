# Myke Cruz x Inky EDC

A React site for Myke Cruz's leather EDC wallets and attachments.

## Scripts

- `npm start` — run the dev server at [http://localhost:3000](http://localhost:3000)
- `npm run build` — build for production into `build/`

## Structure

```
src/
  App.js / App.css     — root component, routes, global styles
  index.js / index.css — entry point
  components/          — shared UI (nav/footer, icons, patch graphic)
  pages/                — route-level pages (Home, Cruzer Models, Cruzer Attachments)
```
